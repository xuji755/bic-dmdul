# DM8 Research Fixtures

These SQL files create controlled states for byte-level storage research. They
are calibration fixtures, not production extractor inputs.

Run them on a disposable DM8 instance. After each fixture phase, capture:

- online reference query output;
- relevant dictionary rows;
- clean or snapshot copies of data files;
- raw page evidence generated by `dmdul capture-evidence`.

The extractor must not depend on online views. Online output is used only to
label and verify raw-byte observations.
